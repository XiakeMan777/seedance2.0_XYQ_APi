📋 任务管理

POST /api/generate-video
提交视频生成任务（multipart/form-data）
参数类型必填说明
promptstring✅视频描述提示词
durationint✅时长：5 / 10 / 15 秒
ratiostring✅比例：16:9 / 9:16
modelstring✅seedance-2.0-fast / seedance-2.0
filesfile[]✅参考图片（至少1张）
{ "status": "success", "task_id": "uuid", "message": "视频生成任务已提交", "required_credits": 50, "running_tasks": 1 }
GET /api/task/{task_id}
查询任务状态
{ "task_id": "uuid", "prompt": "...", "status": "running", "progress": 45, "video_path": null }
POST /api/task/{task_id}/retry
重试失败的任务
DELETE /api/task/{task_id}
删除任务（运行中的任务无法删除）
GET /api/tasks?status=&limit=100&offset=0
列出所有任务（支持按状态筛选和分页）
POST /api/tasks/clear
清空所有任务（运行中的任务不会被清空）

🤖 OpenAI 兼容 API

完全兼容 OpenAI API 格式，可直接对接现有客户端和工具链。

GET /v1/models
获取可用模型列表
{ "object": "list", "data": [{ "id": "seedance-2.0-fast", "object": "model" }, { "id": "seedance-2.0", "object": "model" }] }
POST /v1/videos/generations
提交视频生成任务（同步/异步均可）
参数类型必填说明
modelstring否seedance-2.0-fast / seedance-2.0
promptstring✅视频描述
durationint否5 / 10 / 15 秒（默认10）
ratiostring否16:9 / 9:16（默认16:9）
filesfile[]✅参考图片（multipart）
file_pathsstring[]✅图片URL数组（JSON模式）
{ "created": 1700000000, "task_id": "uuid", "status": "processing", "message": "任务已提交" }
GET /v1/videos/generations/{task_id}
查询任务状态（OpenAI 格式）
{ "created": 1700000000, "task_id": "uuid", "status": "succeeded", "data": [{ "url": "/api/video/uuid", "revised_prompt": "提示词" }] }
POST /v1/videos/generations/async
异步提交任务（立即返回 task_id）
参数与 /v1/videos/generations 完全一致
GET /v1/videos/generations/async/{task_id}?timeout=1200
查询异步任务结果（服务端阻塞等待，最长 20 分钟）
无需客户端轮询，服务端自动等待至任务完成
{ "created": 1700000000, "task_id": "uuid", "status": "succeeded", "data": [{ "url": "/api/video/uuid", "revised_prompt": "提示词" }] }

💡 调用示例

cURL 示例（multipart 上传）：
curl -X POST http://localhost:8033/v1/videos/generations \
  -F "model=seedance-2.0-fast" \
  -F "prompt=@1 图片中的人物开始微笑" \
  -F "ratio=16:9" \
  -F "duration=10" \
  -F "files=@image1.jpg" \
  -F "files=@image2.jpg"
cURL 示例（JSON + 图片URL）：
curl -X POST http://localhost:8033/v1/videos/generations \
  -H "Content-Type: application/json" \
  -d '{
    "model": "seedance-2.0-fast",
    "prompt": "@1 图片中的人物开始微笑",
    "ratio": "16:9",
    "duration": 10,
    "file_paths": ["https://example.com/image1.jpg"]
  }'
异步查询示例：
# 1. 提交任务（立即返回）
curl -X POST http://localhost:8033/v1/videos/generations/async \
  -F "model=seedance-2.0-fast" \
  -F "prompt=描述" \
  -F "files=@image.jpg"

# 2. 查询结果（服务端阻塞等待）
curl http://localhost:8033/v1/videos/generations/async/{task_id}

🔧 调试模式说明

调试模式 是一种特殊运行模式，用于在没有真实 AI 视频生成能力时，测试和验证整个视频生成工作流。

开启后会发生什么？

  • ✅ 跳过所有 AI 模型调用
  • ✅ 跳过真实视频生成流程
  • ✅ 直接返回本地示例视频
  • ✅ 快速完成整个任务流程

适用场景？

  • 🚀 前端界面功能测试
  • 🚀 视频生成流程调试
  • 🚀 在没有积分/Cookie 时演示
  • 🚀 开发阶段快速验证

关闭后呢？

  • 🔄 恢复正常视频生成流程
  • 🔄 需要有效的 Cookie 和积分
  • 🔄 会真实调用剪映 AI 服务

💡 提示：调试模式开启后，所有新建任务都会使用本地示例视频，而不会消耗积分。
